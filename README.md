# nvim-config
Minimal configuration for nvim.
- Oil (File explorer)
- mini.pick (File picker)
  - Icons
- nvim-lspconfig (Config for lsps)
- mason (Install lsps)
- rustaceanvim (Rust lsp config for rust-analyzer)
- typst-preview (Support for typst)
- QoL pluggins

## For rust support install:

- rust-src
- rust
- rustfmt (optional, recommended)
